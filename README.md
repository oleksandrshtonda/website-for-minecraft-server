# Website for a Minecraft server!

Economy, Governments, Messages and other!

## Frontend Tech stack:
 - React
 - Zustand
 - socket.io-client
 - Axios
 - React router dom
 - TypeScript

## Backend Tech stack:
 - NestJS
 - NestJS/jwt
 - NestJS/Passport
 - Bcrypt
 - Platform-socket.io
 - Minecraft-protocol
 - PostgreSQL
 - TypeORM

## Env:
 - Node.js 22.11.0
 - NPM 10.9.0
